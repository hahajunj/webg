⚡ Game Controls & Tips

Essential Controls:

  • WASD / ⌥W⌥A⌥S⌥D - Move your character
  • Mouse - Aim your bow
  • Left Click - Shoot arrows
  • Spacebar - Jump
  • Q key / Mouse Wheel - Switch between weapons
  • 1-3 keys - Select specific weapons
  • ESC key - Access game menu
  • ALT key / ⌥ Option - Free cursor

Pro Tips:

  • • Jump while shooting for better accuracy from high ground
  • • Use ALT key (Windows) or ⌥ Option (Mac) to toggle mouse between game and browser
  • • Learn map shortcuts for tactical advantages
  • • Coordinate with teammates using squad voice chat

About The Game

Battle players around the world with your bow and arrow in this exciting medieval-themed multiplayer game. Defend your castle or steal the flag with your teammates to win the battle. With 17 unique maps and hidden shortcuts, this 5v5 team-based archery game offers the perfect blend of strategy and action.

💥 GAME FEATURES

  • • Team-Based Combat: Strategic 5v5 matches with friends or global players
  • • Economy System: Earn coins, purchase weapons and equipment
  • • Squad System: Team up with friends using invitation codes
  • • Customization: Over 200 unlockable items and weapon skins

👥 SQUAD SYSTEM

Team up with friends and dominate the battlefield together:

  1. 1. Press ESC and select SQUAD from the menu
  2. 2. Generate or enter an invitation code
  3. 3. Share your code via chat or messaging apps
  4. 4. Play together on the same team with voice chat

Pro Tip: Full squads earn 10% more coins and can create private matches with custom rules!
